#### Euler angles

https://en.wikipedia.org/wiki/Euler_angles: 
Any target orientation can be reached, starting from a known reference orientation, using a specific sequence of intrinsic rotations, whose magnitudes are the Euler angles of the target orientation. This example uses the z-x′-z″ sequence.

#### Universal multiport interferometer

https://en.wikipedia.org/wiki/Universal_multiport_interferometer
This is an example of a set of intrinsic Euler rotations about a rotating coordinate system. Notice that the first and third rotation axes overlap. This animation was made by Juansempere.
